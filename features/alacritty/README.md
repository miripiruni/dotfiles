# Alacritty

A cross-platform, OpenGL terminal emulator. Currently is Beta.

Site: khttps://alacritty.org

## Setup

My Alacritty config depends on Fira Code Font. It will be automaticaly installed
and configured as well.  Just run:

```console
./setup.sh
```

## Uninstall

```console
./uninstall.sh
```
